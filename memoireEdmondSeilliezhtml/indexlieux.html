Index des lieux

    1. Amanvillers
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: 49° 10′ 03″ nord, 6° 02′ 37″ est
    • Description : Village où Edmond et son bataillon prennent le train pour rejoindre la ligne de front.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 5, 5.
    2. Bertrix
    • Pays: Belgique.
    • Département ou province: Province du Luxembourg
    • Coordonnées géographiques: 49° 51,25′ nord, 5° 15,2′ est
    • Description : Etape du train conduisant Edmond captif vers Trèves
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 17.
    3. Budling
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: 49° 20′ 51″ nord, 6° 21′ 23″ est
    • Description : Localité proche d'Inglange. Le bataillon d'Edmond se cache dans un bois proche en mai 1940.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 4.
    4. Canal de l'Aisne
    • Pays: France.
    • Département ou province: Aisne et Marne
    • Coordonnées géographiques: Non précisé
    • Description : Canal de 58 km reliant l'Aisne à la Marne.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 7.
    5. Capsilliers
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: indéterminé
    • Description : Toponyme non élucidé. Il s'agit probablement d'un lieu-dit dont l'emplacement est indéterminé.
    • Attention, l'emplacement de ce toponyme est incertain.
    6. Charleville-Mézières
    • Pays: France.
    • Département ou province: Ardennes
    • Coordonnées géographiques: 49° 46′ 19″ nord, 4° 42′ 58″ est
    • Description : Lieu de cantonnement d'Edmond, au sein de la caserne des gardes-mobiles.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 16.
    7. Colmar
    • Pays: France.
    • Département ou province: Haut-Rhin
    • Coordonnées géographiques: 48° 04′ 54″ nord, 7° 21′ 20″ est
    • Description : Mentionné par Edmond lors du remplacement du 152e Régiment d'Infanterie
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 7.
    8. Elincourt
    • Pays: France.
    • Département ou province: Nord
    • Coordonnées géographiques: 50° 02′ 42″ nord, 3° 22′ 21″ est
    • Description : Village rural à la frontière administrative entre les département du Nord, de l'Aisne et des Ardennes, expliquant vraisemblablement l'approximation d'Edmond qui le situe dans les Ardennes (à comprendre sûrement sous la plume de l'auteur comme "région" historique ou géographique).
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 7, 7.
    9. Epernay
    • Pays: France.
    • Département ou province: Marne
    • Coordonnées géographiques: 49° 02′ 25″ nord, 3° 57′ 36″ est
    • Description : Ville mentionnée par Edmond comme repère géographique pour situer le village X dont il ne parvient pas à se souvenir.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes :
    10. Inglange
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: 49° 20′ 52″ nord, 6° 17′ 56″ est
    • Description : Ville de Moselle où Edmond et son bataillon stationne 10 jours en mai 1940.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 3, 3, 3.
    11. Kirschnaumen
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: 49° 23′ 54″ nord, 6° 26′ 06″ est
    • Description : Commune de cantonnement d'Edmond en avril 1940
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 2.
    12. Metzervisse
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: 49° 18′ 57″ nord, 6° 17′ 11″ est
    • Description : Localité proche de Budling et Inglange.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 4, 4.
    13. Pernes-en-Artois
    • Pays: France.
    • Département ou province: Pas-de-Calais
    • Coordonnées géographiques: 50° 29′ 06″ nord, 2° 24′ 41″ est
    • Description : Bourg natal d'Edmond Seilliez.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 1, 2, 19.
    14. Perthes
    • Pays: France.
    • Département ou province: Haute-Marne
    • Coordonnées géographiques: 48° 39′ 25″ nord, 4° 49′ 27″ est
    • Description : Ville a proximité de laquelle le bataillon d'Edmond est capturé lors de leur retraite.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 13, 13, 15.
    15. Prusse Orientale
    • Pays: Allemagne.
    • Département ou province: Province de Prusse-Orientale
    • Coordonnées géographiques: Non précisé
    • Description : Province faisant théorique partie de l'Etat Libre de Prusse, elle est occupée et administrée par l'Allemagne en 1940. Territoire de transfert des soldats captifs entre 1940 et début 1941.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 19.
    16. Reims
    • Pays: France.
    • Département ou province: Marne
    • Coordonnées géographiques: 49° 15′ 46″ nord, 4° 02′ 05″ est
    • Description : Mentionné par Edmond comme direction générale empruntée par son bataillon lors de sa retraite.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 13.
    17. Rethel
    • Pays: France.
    • Département ou province: Ardennes
    • Coordonnées géographiques: 49° 30′ 38″ nord, 4° 22′ 00″ est
    • Description : Mentionné à plusieurs reprise. Espace des combats et des bombardements que relate Edmond. Il s'agit parfois de la ville précisement, parfois d'un espace géographique moins clairement circonscrit.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 7, 7, 7, 9, 10, 10, 16.
    18. Sainte-Marie-aux-Chênes
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: 49° 11′ 32″ nord, 6° 00′ 11″ est
    • Description : Mentionné par Edmond comme étant une localité qu'il a déjà visité avant les combats.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 5.
    19. Semécourt
    • Pays: France.
    • Département ou province: Moselle
    • Coordonnées géographiques: 49° 11′ 38″ nord, 6° 08′ 07″ est
    • Description : Etape du parcours du bataillon d'Edmond en Moselle. L'auteur précise la région entre parathèses. Cela traduit vraisemblablement une volonté de désambiguiser le nom du lieu.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 4, 5.
    20. Trèves (Trier)
    • Pays: Allemagne.
    • Département ou province: Rhénanie-Palatinat
    • Coordonnées géographiques: 49° 45′ 35″ nord, 6° 38′ 38″ est
    • Description : Edmond mentionne le nom allemand de la ville en plus du nom français (probablement une retranscription orale).
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 17, 19.
    21. Vitry-le-François
    • Pays: France.
    • Département ou province: Marne
    • Coordonnées géographiques: 48° 43′ 32″ nord, 4° 35′ 07″ est
    • Description : Gare par laquelle devaient transiter l'artillerie lourde française, qui fut d'après Edmond bombardée peu avant son passage.
    • L'emplacement de ce toponyme est certain.
    • Occurences aux pages suivantes : 6.
